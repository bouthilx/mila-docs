Mila documentation¶

This documentation aims to cover the information required to run scientific and data-intensive computing tasks at Mila and the available resources for its members. It is targeted towards Mila researchers and students.

Support

To reach the Mila infrastructure support, please file a ticket at https://it-support.mila.quebec/

Contribution

If you find any error in the documentation, missing or unclear sections, or would simply like to contribute, please open an issue or make a pull request on the github page.
